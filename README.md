# homebrew-tap

Personal Homebrew tap for software by [Sudhanshu Ambastha](https://github.com/Sudhanshu-Ambastha).

## Setup

Add this tap once:

```bash
brew tap sudhanshu-ambastha/tap
```

## Installing

### jar-cart

Lightning-fast, no-build Java build orchestrator — fetch, cache, and run Java artifacts instantly without Maven or Gradle.

```bash
brew install jar-cart
```

This installs both the `jar-cart` command and its shorthand alias `jc`.

- Repo: [Sudhanshu-Ambastha/jar-cart](https://github.com/Sudhanshu-Ambastha/jar-cart)
- Docs & usage: see the [jar-cart README](https://github.com/Sudhanshu-Ambastha/jar-cart#readme)

## Uninstalling

```bash
brew uninstall jar-cart
brew untap sudhanshu-ambastha/tap
```
